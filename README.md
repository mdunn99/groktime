# GrokTime - Dynamic Grokker

GrokTime parses log lines with Grok patterns and can learn new patterns on-demand using OpenAI.

## Core Features
- A preset list of common Grok patterns in common log formats (generated by GPT-5-mini)
- Neglgible API cost for writing new Grok patterns
- Configurable variable names
- Usable as a Python library and terminal entrypoint.

## Dependencies
- `pygrok`
- `openai`
- `watchdog`

## Install
1. Clone this repo: `git clone https://github.com/mdunn99/groktime.git`
2. Create/activate a virtual environment
3. Create a `.env` file with your OpenAI API key: `OPENAI_API_KEY=<YOUR_KEY_HERE>`
4. Install the dependencies: `pip install pygrok openai watchdog`

## Library Usage
```python
from groktime import EventHandler, LogProcessor, run_observer

# Option 1: run file observer loop
run_observer(log="/var/log/auth.log", output="out.json", grok_patterns_file="patterns.json")

# Option 2: process lines directly
processor = LogProcessor(output="out.json", grok_patterns_file="patterns.json")
processor.process([
    "Aug 31 12:34:56 myhost sshd[123]: Accepted password for alice from 10.0.0.2 port 2222"
])
```

## CLI Usage
```bash
python3 main.py -l /path/to/logfile -o out.json -p patterns.json
```
